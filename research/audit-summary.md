**360-degree approach:** On-track athletic training + off-track academic support + life-skills mentoring.

## 13. Key Problems (from client brief + our audit)

### Website problems
1. WordPress + Elementor = constant plugin updates, admin burden
2. Staff hate managing it
3. Contact page still has Lorem ipsum placeholder
4. Holiday programmes page is one paragraph
5. Registration form (Upshot) is too long and confusing
6. Impact page has no live data
7. Team page shows images but no names/roles
8. Not optimised for the 3 ICP journeys (parents, schools, funders)

### Operational problems
1. All registration data manually entered from paper to Upshot
2. Attendance tracked manually
3. Outcomes tracked manually
4. No live impact dashboard for funders
5. Hosting renewal £160/year seen as pricey

### CEO perception problems
1. Thinks a system like this would cost £10k+
2. Doesn't quite understand the simplicity of the solution
3. Values Upshot highly ("unbeatable" — it isn't)

## 16. Opportunities for the Rebuild

### Must-haves
1. Clean, responsive website that speaks to 3 ICPs (parents, schools, funders)
2. Simplified registration flow (multi-step wizard, save progress, sibling autofill)
3. Upshot integration (API or data sync layer)
4. Impact dashboard with live data (attendance, outcomes, demographics)
5. Events listing that's database-driven (not manually edited)
6. Fixtures page that's database-driven with filters
7. SEO-optimised pages for Brent/local keywords
8. Contact form with audience routing (not "Lorem ipsum")
