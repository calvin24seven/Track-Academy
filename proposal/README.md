# Proposal — Track Academy Website & Systems Rebuild

Single-page scroll-based proposal for Patrik Ewe (CEO), modelled on the Mindbeat proposal structure.

## Files
- `index.html` — self-contained proposal page (HTML + CSS + JS embedded)
- `assets/track-academy-logo.png` — colour logo (for light backgrounds)
- `assets/track-academy-logo-white.png` — white logo (for dark backgrounds, used on cover)

## To view locally
Open `index.html` in a browser.

## To deploy
Upload the entire `proposal/` folder to a Cloudflare Worker or any static host (like the Mindbeat proposal at https://mindbeat-proposal.calvin-6fc.workers.dev/).

## Structure (13 sections)
1. Cover — Track Academy logo, hero, prepared for Patrik Ewe
2. What we heard — four observations from discovery
3. The vision — old perception vs new direction table
4. Three audiences — Parents, Schools, Funders cards
5. Site diagnosis — 8 specific problems to fix
6. New architecture — sitemap
7. Homepage direction — before/after
8. Registration reimagined — today vs rebuild comparison
9. Impact dashboard — stat band mock-up
10. Integrations — Upshot, Spond, OpenTrack, Whydonate
11. Search opportunity — keyword clusters
12. Roadmap — 4 phases
13. Investment — £800 rebuild + £300/mo retainer + £160 hosting saved
14. Next step — closing
